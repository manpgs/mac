NAME

photosctl — Photo Library diagnostic, configuration and test utility

SYNOPSIS

photosctl [<command> ...]

DESCRIPTION

photosctl supports various diagnostic, configuration, test and other operations on Photo Libraries.

New style photosctl commands can support options.

Most newer commands operate on a Photo Library. These commands all support the option --library <path>, or will operate on the System Photo Library if one is set.

Legacy commands operate only on the System Photo Library.
