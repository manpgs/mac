NAME

hdiejectd — disk image management daemon

SYNOPSIS

hdiejectd

DESCRIPTION

hdiejectd manages disk images attached to the system, and arbitrates the image attach and eject processes. hdiejectd will run on demand and automatically exit after a short time when there are no longer any disk images attached to the system.

There are no configuration options to hdiejectd. Users should not run hdiejectd manually.

SEE ALSO

hdiutil(1), diskutil(8)
