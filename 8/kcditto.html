NAME

kcditto — macOS kernel management helper tool

DEPRECATED

Please use bless.

DESCRIPTION

kcditto is a helper tool for the staging of kext collection files. It looks for kext collection files on volume where the kcditto binary exists, and copies all the boot kext collections into the preboot volume where efiboot can find them.

FILES

/usr/local/sbin/kcidtto
