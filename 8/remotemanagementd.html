NAME

remotemanagementd — Remote Management protocol daemon

SYNOPSIS

remotemanagementd

DESCRIPTION

remotemanagementd handles HTTP communication with a Remote Management server, processing configuration information, and sending status messages back to the server.

FILES

/System/Library/LaunchDaemons/com.apple.remotemanagementd.plist

DIAGNOSTICS

Logging can be viewed via: log stream --predicate 'senderImagePath contains "RemoteManagement.framework"'
