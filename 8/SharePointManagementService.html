NAME

SharePointManagementService — Service agent which manages Share Points

DESCRIPTION

Share Points are directories which are shared with other machines over the network. They are managed by the user in System Preferences > Sharing > File Sharing. SharePointManagementService is the service component that is responsible for the creation and removal of Share Points entries.
