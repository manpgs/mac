NAME

AuthenticationServicesAgent — Authentication Services App Login Agent

DESCRIPTION

AuthenticationServicesAgent is the system launch agent that retrieves credentials for app login requests made with AuthenticationServices API.

There are no configuration options to AuthenticationServicesAgent, and users should not run AuthenticationServicesAgent manually.
