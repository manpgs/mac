NAME

mbbackgrounduseragent — helper for the Setup Assistant application.

DESCRIPTION

mbbackgrounduseragent is used by the Setup Assistant application and runs as the target user in order to configure that user from a background agents or non-Aqua session agents. It is not meant to be invoked directly.
