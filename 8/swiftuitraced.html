SWIFTUITRACED(8) System Manager's Manual SWIFTUITRACED(8)

NAME

swiftuitraced — Collects tracepoints from SwiftUI

SYNOPSIS

takes no arguments, and users should not launch it manually.

DESCRIPTION

This daemon orchestrates the process of collecting and writing tracepoints that are coming from SwiftUI layer. It initializes only during tracing
