NAME

backgroundtaskmanagementagent — User agent which manages lists of background task processes

DESCRIPTION

backgroundtaskmanagementagent is a per-user agent that manages lists of background tasks. This includes items in System Preferences > Users & Groups > Login Items, items installed with SMLoginItemSetEnabled(), and items from managed device profiles.

FILES
