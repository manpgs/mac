APPLEVIRTUALPLATFORMHIDBRIDGE(8) System Manager's Manual APPLEVIRTUALPLATFORMHIDBRIDGE(8)

NAME

AppleVirtualPlatformHIDBridge — Bridge HID services from a Virtualization host

SYNOPSIS

AppleVirtualPlatformHIDBridge takes no arguments, and users should not launch it manually.

DESCRIPTION

When macOS is running as a guest operating system in a virtual machine created by the Virtualization framework and configured with an AppleVirtualPlatformHIDBridge device, AppleVirtualPlatformHIDBridge instantiates and manages virtual input devices on behalf of the host.

HISTORY

AppleVirtualPlatformHIDBridge first appeared in macOS 13.
