NAME

SafariHistoryServiceAgent — Safari History Service Agent

DESCRIPTION

SafariHistoryServiceAgent is the system launch agent that launches Safari's history service to finish clearing history if necessary.

There are no configuration options to SafariHistoryServiceAgent, and users should not run SafariHistoryServiceAgent manually.
