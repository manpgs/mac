NAME

BackgroundTaskManagementAgent — User agent which manages notifications about background task processes

DESCRIPTION

BackgroundTaskManagementAgent is a per-user agent that manages notifications of background tasks. This includes items in System Settings > General > Login Items, items installed with SMAppService or SMLoginItemSetEnabled(), and items from managed device profiles.

This command is not intended to be invoked directly.
