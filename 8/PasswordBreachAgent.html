PASSWORDBREACHAGENT(8) System Manager's Manual PASSWORDBREACHAGENT(8)

NAME

PasswordBreachAgent — Leaked Password Warning Agent

DESCRIPTION

PasswordBreachAgent is the system launch agent that checks if saved website passwords have been seen in a data leak.

There are no configuration options to PasswordBreachAgent, and users should not run PasswordBreachAgent manually.
