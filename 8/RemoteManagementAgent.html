NAME

RemoteManagementAgent — Remote Management protocol agent

SYNOPSIS

RemoteManagementAgent

DESCRIPTION

RemoteManagementAgent handles HTTP communication with a Remote Management server, processing configuration information, and sending status messages back to the server.

FILES

/System/Library/LaunchAgents/com.apple.RemoteManagementAgent.plist

DIAGNOSTICS

Logging can be viewed via: log stream --predicate 'senderImagePath contains "RemoteManagement.framework"'
