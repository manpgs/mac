NAME

enhancedloggingd — A background daemon for Enhanced Logging.

DESCRIPTION

enhancedloggingd manages collecting diagnostic files from users reporting issues to Apple Support.

There are no configuration options for this service. Users should not run enhancedloggingd manually.
