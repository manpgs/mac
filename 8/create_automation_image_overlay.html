NAME

create_automation_image_overlay — overlay automation image content onto another directory

SYNOPSIS

create_automation_image_overlay overlay_content target_dir [rules ...]

DESCRIPTION

create_automation_image_overlay causes the given overlay_content to be appear on top of target_dir using union mounts. It is not intended for general use, and its behavior is not documented.
