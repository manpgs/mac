NAME

parentalcontrolsd — Parental Controls helper process.

SYNOPSIS

parentalcontrolsd

DESCRIPTION

parentalcontrolsd is used by Parental Controls to manage and track restricted users.

NOTE

parentalcontrolsd should not be invoked directly. It is intended only for use by Parental Controls.
