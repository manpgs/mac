SAFARIBOOKMARKSSYNCAGENT(8) System Manager's Manual SAFARIBOOKMARKSSYNCAGENT(8)

NAME

SafariBookmarksSyncAgent — Safari Bookmarks Sync Agent

DESCRIPTION

SafariBookmarksSyncAgent is the system launch agent responsible for syncing Safari Bookmarks, Reading List, and tabs with iCloud via CloudKit.

There are no configuration options to SafariBookmarksSyncAgent, and users should not run SafariBookmarksSyncAgent manually.
