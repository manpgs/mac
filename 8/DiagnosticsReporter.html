NAME

Diagnostics Reporter — Facilitates feedback and diagnostics reporting for Panics and ALEs

DESCRIPTION

Diagnostics Reporter is an application that launches when a panic log is present in /var/db/PanicReporter or when an ALE crash log is present in /var/db/DiagnosticsReporter. It is responsible for prompting the users to submit feedback and diagnostic logs on seed and internal builds. It will not prompt more than twice a day.
