NAME

UnmountAssistantAgent — Provides help when a disk can't be ejected.

DESCRIPTION

UnmountAssistantAgent displays a dialog telling a user which process is preventing a disk eject. If the process can't be quit by the user, it will offer an option to force eject the disk.

April 20, 2017 macOS 12.4
