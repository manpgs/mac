NAME

wifip2pd — Daemon responsible for managing peer-to-peer Wi-Fi discovery and data links.

SYNOPSIS

wifip2pd

DESCRIPTION

wifip2pd is a system-level daemon responsible for configuring the Wi-Fi interface to work over peer-to-peer protocols.

There are no configuration options to wifip2pd, and users should not run wifip2pd manually.
