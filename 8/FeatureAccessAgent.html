NAME

FeatureAccessAgent — iCloud Subscription Status Management

SYNOPSIS

takes no arguments, and users should not launch it manually.

DESCRIPTION

FeatureAccessAgent is a user agent that manages iCloud subscription status.

HISTORY

FeatureAccessAgent first appeared in macOS 15.0.
