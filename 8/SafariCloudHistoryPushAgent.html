NAME

SafariCloudHistoryPushAgent — Safari Cloud History Push Agent

DESCRIPTION

SafariCloudHistoryPushAgent is the system launch agent that listens for changes to Safari history from other devices.

There are no configuration options to SafariCloudHistoryPushAgent, and users should not run SafariCloudHistoryPushAgent manually.
