NAME

SpeechSynthesisServer — Implements speaking hotkey and time announcements text-to-speech features.

DESCRIPTION

SpeechSynthesisServer is a user-level daemon that runs only when the user has enabled the speaking hotkey feature (Accessibility preference pane) or time announcements features (Date & Time preference pane). The daemon will continue running while either feature is enabled.
