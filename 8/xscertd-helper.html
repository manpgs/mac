NAME

xscertd-helper — xscertd helper process

SYNOPSIS

xscertd-helper

DESCRIPTION

The xscertd-helper utility is used by xscertd for various functions.

NOTE

The xscertd-helper utility should not be invoked directly. It is intended only for use by xscertd.
