NAME

NANDTaskScheduler — Daemon to periodically refresh data on NAND

SYNOPSIS

takes no arguments, and users should not launch it manually.

DESCRIPTION

NANDTaskScheduler periodically runs in background to refresh data stored on NAND to improve data retention

HISTORY

first appeared in macOS 13.0
