NAME

diskmanagementstartup — DiskManagement.framework helper tool

SYNOPSIS

diskmanagementstartup

DESCRIPTION

diskmanagementstartup is a tool that runs at startup to perform certain cleanup and configuration actions related to or armed by the DiskManagement framework and its diskmanagementd daemon. This tool is itself a DiskManagement client.

There are no configuration options. Users should not run this program manually.

SEE ALSO

diskmanagementd(8), diskarbitrationd(8), diskutil(8), mount(8)
